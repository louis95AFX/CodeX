Pandas DataFrames Basics

Pandas is the fundamental package for data analysis in Python. It provides high-performance, easy-to-use data structures and data analysis tools.

Key Pandas Concepts

  • Series - One-dimensional labeled array capable of holding any data type
  • DataFrame - Two-dimensional labeled data structure with columns of potentially different types
  • Index - Immutable sequence used for indexing and alignment
  • GroupBy - Split-apply-combine operations on DataFrames

Real-World Example

Imagine analyzing sales data:

  • Load CSV data into a DataFrame
  • Clean missing values
  • Group by product category
  • Calculate summary statistics
  • Visualize trends over time

Need a hint?

For numerical columns, use fillna() with df[column].mean(). For categorical columns, fill with a string like "Unknown". Use pd.to_datetime() with errors='coerce' to handle invalid dates.

def clean_data(df):
    # Fill numerical missing values with column mean
    num_cols = df.select_dtypes(include=['number']).columns
    df[num_cols] = df[num_cols].fillna(df[num_cols].mean())
    
    # Fill categorical missing values with "Unknown"
    cat_cols = df.select_dtypes(exclude=['number']).columns
    df[cat_cols] = df[cat_cols].fillna("Unknown")
    
    # Convert date column to datetime
    df['date'] = pd.to_datetime(df['date'], errors='coerce')
    
    return df
