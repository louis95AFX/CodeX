Introduction to SQL Databases

SQL (Structured Query Language) is the standard language for managing and manipulating relational databases. It allows you to create, read, update, and delete data (CRUD operations) in a structured way.

Why Learn SQL?

  • Used by 99% of relational databases worldwide
  • Essential skill for data analysts, developers, and DBAs
  • Standardized language across database systems
  • High demand in the job market
  • Foundation for understanding data structures

Database Management Systems

MySQL

  • Open-source RDBMS
  • Popular for web applications
  • Owned by Oracle

PostgreSQL

  • Advanced open-source RDBMS
  • Supports complex queries
  • Highly extensible

SQL Server

  • Microsoft's enterprise RDBMS
  • Integrated with .NET ecosystem
  • Powerful BI tools

Basic SQL Commands

Data Definition Language (DDL)

  • CREATE - Create database objects
  • ALTER - Modify database objects
  • DROP - Delete database objects
  • TRUNCATE - Remove all records

Data Manipulation Language (DML)

  • SELECT - Retrieve data
  • INSERT - Add new records
  • UPDATE - Modify records
  • DELETE - Remove records

Sample Database Schema

We'll use a simple e-commerce database with these tables:

Table Description
customers Customer information
products Product catalog
orders Customer orders
order_items Items within each order

Creating a Table

Here's how to create the customers table:

CREATE TABLE customers (
    customer_id INT PRIMARY KEY,
    first_name VARCHAR(50) NOT NULL,
    last_name VARCHAR(50) NOT NULL,
    email VARCHAR(100) UNIQUE,
    join_date DATE DEFAULT CURRENT_DATE,
    is_active BOOLEAN DEFAULT TRUE
);

Basic SELECT Query

Retrieve data from the products table:

SELECT product_id, product_name, price
FROM products
WHERE price BETWEEN 10 AND 100
ORDER BY price DESC
LIMIT 10;

Pro Tip

Always include a WHERE clause to filter your results. Retrieving all rows with SELECT * can be inefficient for large tables.
