Introduction to MQL5

MQL5 (MetaQuotes Language 5) is a high-level programming language designed for developing trading robots (Expert Advisors), technical indicators, scripts, and libraries in the MetaTrader 5 trading platform.

Why Learn MQL5?

  • Automate your trading strategies
  • Create custom technical indicators
  • Backtest trading ideas quickly
  • Join the algorithmic trading revolution
  • Potential income from selling EAs

MQL5 vs MQL4

MQL5 Improvements

  • Object-oriented programming support
  • More built-in technical indicators
  • Enhanced strategy tester
  • Direct database access
  • Network functions

Similarities

  • Similar basic syntax
  • Same development environment
  • Compatible with MetaTrader platform
  • Common trading functions

Your First Expert Advisor

Here's a simple "Hello World" Expert Advisor that displays a message when attached to a chart:

//+------------------------------------------------------------------+
//|                                                      HelloEA.mq5 |
//|                        Copyright 2023, MetaQuotes Software Corp. |
//|                                             https://www.metaquotes.net/ |
//+------------------------------------------------------------------+
#property copyright "Copyright 2023, MetaQuotes Software Corp."
#property link      "https://www.metaquotes.net/"
#property version   "1.00"

//+------------------------------------------------------------------+
//| Expert initialization function                                   |
//+------------------------------------------------------------------+
int OnInit()
  {
   Alert("Hello, MQL5 World!");
   return(INIT_SUCCEEDED);
  }

MQL5 Program Structure

MQL5 programs have a specific structure depending on their type (EA, Indicator, Script):

Expert Advisor

  • OnInit() - Initialization
  • OnDeinit() - Deinitialization
  • OnTick() - Main trading logic
  • OnTimer() - Timer events
  • OnChartEvent() - UI events

Indicator

  • OnInit() - Initialization
  • OnDeinit() - Deinitialization
  • OnCalculate() - Main calculation
  • Indicator buffers
  • Plotting functions

Script

  • OnStart() - Main function
  • Single execution
  • No event handling
  • Good for one-time tasks
