Text Formatting

Learn how to style and format text using CSS properties like text-align, text-decoration, and more.

4.0
30 min 3 exercises

CSS Text Properties

CSS provides numerous properties for styling text. These properties allow you to control the appearance of text, including alignment, decoration, spacing, and more.

Key Text Properties:

  • text-align - Aligns text horizontally
  • text-decoration - Adds underlines, overlines, etc.
  • text-transform - Changes text capitalization
  • letter-spacing - Adjusts space between characters
  • line-height - Sets the height of text lines
  • text-shadow - Adds shadow effects to text

Text Alignment

The text-align property specifies the horizontal alignment of text:

p {
  text-align: center; /* left | right | justify */
}

text-align: left;

Left aligned text

text-align: center;

Centered text

text-align: right;

Right aligned text

text-align: justify;

Justified text fills the width by adjusting word spacing.

Text Decoration

The text-decoration property adds decoration to text:

a {
  text-decoration: none; /* Removes underline from links */
}

h1 {
  text-decoration: underline wavy red; /* Multiple values */
}

text-decoration: underline;

Underlined text

text-decoration: overline;

Overlined text

text-decoration: line-through;

Line through text

text-decoration: underline dotted;

Dotted underline

Best Practice

Use text decorations sparingly. Overusing them can make your content harder to read. Also, remember that underlines are typically associated with links, so use alternative decorations for emphasis.
