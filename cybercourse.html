Understanding Basic Security Concepts

Cybersecurity is the practice of protecting systems, networks, and programs from digital attacks. These cyberattacks are usually aimed at accessing, changing, or destroying sensitive information; extorting money from users; or interrupting normal business processes.

The CIA Triad

The foundation of cybersecurity lies in three core principles known as the CIA triad:

  • Confidentiality - Ensuring that information is accessible only to those authorized to have access
  • Integrity - Safeguarding the accuracy and completeness of information and processing methods
  • Availability - Ensuring that authorized users have access to information and associated assets when required

Real-World Example

Imagine an online banking system:

  • Confidentiality: Only you should be able to see your account balance
  • Integrity: No one should be able to alter your transaction amounts
  • Availability: You should be able to access your account whenever you need

Quick Quiz

Which CIA triad component is violated when a hacker prevents legitimate users from accessing a website?

Correct! This is a classic example of a Denial of Service (DoS) attack that violates availability.

Need a hint?

Try converting each character to its ASCII code using charCodeAt(), apply the shift (remember to handle wrap-around from Z to A), then convert back to a character with fromCharCode(). Don't forget to handle lowercase and uppercase separately!

function caesarEncrypt(plaintext, shift) {
    return plaintext.split('').map(char => {
        if (/[a-z]/.test(char)) {
            return String.fromCharCode(((char.charCodeAt(0) - 97 + shift) % 26) + 97);
        }
        if (/[A-Z]/.test(char)) {
            return String.fromCharCode(((char.charCodeAt(0) - 65 + shift) % 26) + 65);
        }
        return char;
    }).join('');
}
