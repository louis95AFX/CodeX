Introduction to Python

Python is a powerful, versatile programming language that's perfect for beginners and professionals alike. In this course, you'll learn Python from the ground up, starting with the absolute basics.

Why Learn Python?

  • Simple and readable syntax
  • Versatile (web, data science, AI, automation, etc.)
  • Large community and extensive libraries
  • High demand in the job market

Your First Python Program

Traditionally, the first program you write in any language is "Hello, World!". In Python, this is incredibly simple:

>>> print("Hello, World!")
Hello, World!

Python Syntax Basics

Python is known for its clean, readable syntax. Here are some key features:

Indentation

Python uses indentation to define code blocks instead of curly braces.

if 5 > 2:
print("Five is greater than two!")

Comments

Use # for single-line comments and triple quotes for multi-line comments.

# This is a comment
"""
This is a
multi-line comment
"""

Variables

Variables are created when you assign a value to them.

x = 5
name = "John"

Data Types

Python has various data types built-in by default.

# Integer
x = 5
# String
name = "John"
# Boolean
is_active = True

Python Interactive Shell

Python comes with an interactive shell where you can execute commands immediately. Try these examples:

>>> 5 + 3
8
>>> "Hello" + " " + "World"
'Hello World'
>>> len("Python")
6

Pro Tip

Use the type() function to check the data type of any variable. For example: type(5) returns <class 'int'>
