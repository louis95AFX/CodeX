Creating Your First React Native App

React Native lets you build mobile apps using only JavaScript. It uses the same design as React, letting you compose a rich mobile UI from declarative components.

Key Concepts

  • Components - Building blocks of React Native apps
  • JSX - Syntax extension for JavaScript that looks like HTML
  • Props - Short for properties, used to pass data between components
  • State - Used to manage data that changes over time

Real-World Example

A simple counter app demonstrates:

  • Creating functional components
  • Using state with useState hook
  • Styling with StyleSheet
  • Handling user interactions

Quick Quiz

Which command is used to create a new React Native project?

Correct! npx react-native init is the correct command to create a new React Native project.

  const [count, setCount] = useState(0);
