Introduction to SQL

SQL (Structured Query Language) is a standard programming language specifically designed for managing and manipulating relational databases.

Key Features of SQL:

  • Used to communicate with databases
  • Standardized language (ANSI/ISO)
  • Used by major database systems (MySQL, PostgreSQL, SQL Server, Oracle)
  • Allows you to create, read, update, and delete database records (CRUD operations)

Did You Know? SQL was first developed at IBM in the 1970s and became commercially available in 1979. Despite being over 40 years old, it remains the most widely used database language today.

Common SQL Commands:

Data Query Language (DQL)

SELECT - Retrieves data from database

Data Definition Language (DDL)

CREATE, ALTER, DROP - Defines database structure

Data Manipulation Language (DML)

INSERT, UPDATE, DELETE - Modifies data

Data Control Language (DCL)

GRANT, REVOKE - Controls access
