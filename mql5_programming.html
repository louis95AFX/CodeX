Lesson 3

Variables & Data Types in MQL5

8 min 15 XP

Variables are the building blocks of any MQL5 program. They allow you to store and manipulate data in your Expert Advisors, indicators, and scripts.

Basic Data Types

MQL5 supports several fundamental data types:

  • int - integer numbers (e.g., 42, -15)
  • double - floating-point numbers (e.g., 3.14, -0.001)
  • bool - boolean values (true or false)
  • string - text (e.g., "Hello, MQL5!")
  • datetime - date and time (e.g., D'2023.12.31 23:59:59')
  • color - color values (e.g., clrRed, C'255,0,0')

Declaring Variables

To declare a variable in MQL5, specify its type followed by the variable name:

int orderCount = 0;
double currentPrice = "SymbolInfoDouble"("EURUSD", SYMBOL_BID);
bool isTrendUp = true;
string greeting = "Welcome to MQL5 programming!";
datetime expiration = D'2023.12.31';
color buyColor = clrBlue;

Pro Tip

Always initialize your variables when declaring them. This prevents unexpected behavior in your trading algorithms.

Variable Scope

Variables in MQL5 can have different scopes:

  • Local variables - declared inside functions and only accessible within that function
  • Input variables - declared with the input modifier, visible in EA properties
  • Global variables - declared outside functions, accessible throughout the program
// Global variable
int globalCounter = 0;

// Input parameter (visible in EA settings)
input int MovingAveragePeriod = 14;

void OnStart()
{
    // Local variable
    double currentPrice = "SymbolInfoDouble"("EURUSD", SYMBOL_BID);
    
    // ... rest of the code
}

Type Conversion

MQL5 provides both implicit and explicit type conversion:

int integerValue = 10;
double doubleValue = integerValue; // Implicit conversion

string textValue = "123.45";
double numericValue = (double)textValue; // Explicit conversion

Trading Application

When working with trading functions, you'll often need to convert between types. For example, converting a string symbol name to use in trading functions, or converting price values between double and string for display purposes.
